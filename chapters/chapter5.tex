\chapter{面向碳纳米管阵列研究的科研辅助分析平台设计与实现}
\enchapter{Design and Implementation of a Scientific Research Assistant Analysis Platform for Carbon Nanotube Array Research}
\label{chap:platform_design}

\section{引言}
\ensection{Introduction}

前两章已经分别完成了面向碳纳米管阵列研究的知识建模与检索增强、关键形貌自动化表征以及工艺—形貌关系分析。第三章对应全文分析链路中的知识支撑层，主要解决文献、实验记录和分析结果的组织与检索问题；第四章对应核心分析层，主要解决样品 SEM 图像中的关键形貌提取以及工艺参数驱动的关系分析问题。这些工作为样品分析、工艺比较和结果解释奠定了方法基础。

在实际研究过程中，知识查询、SEM 图像分析、工艺参数比较和结果解释通常围绕同一样品或同一组工艺条件连续展开。研究人员往往需要同时调取实验记录、图像表征结果、领域知识和预测信息。若前述方法仅以独立算法或离线流程存在，则难以支撑完整的科研分析活动。因此，有必要将相关方法集成为统一平台，形成面向碳纳米管阵列研究的协同分析流程。

本章在前文研究基础上，围绕科研辅助分析平台的设计与实现展开。首先介绍平台架构与运行环境，随后说明知识增强、自动化形貌表征和形貌预测等方法在平台中的集成方式，最后结合知识增强分析、SEM 自动化表征以及形貌预测与解释等场景，对平台功能进行验证。

\section{系统设计与运行环境}
\ensection{System Design and Runtime Environment}

\subsection{系统需求分析}
\ensubsection{System Requirement Analysis}

面向碳纳米管阵列研究的科研辅助分析平台，其需求主要来自样品分析、工艺比较和结果解释等实际研究活动。平台需要以样品编号为主线，对样品信息、工艺参数、SEM 图像、知识单元和分析结果进行统一管理，使同一样品相关的多源数据能够被连续调用。与此同时，文献、实验记录和历史分析结果需要经过整理、构建与入库后，才能为知识增强分析提供稳定的数据基础。平台还应在同一环境中集成知识增强分析、SEM 自动化表征和形貌预测三类核心任务，减少跨工具切换带来的割裂，并保证各类分析结果能够相互关联。对于知识检索结果、自动化表征指标和预测输出，还需要保留对应的样品、工艺条件和来源证据，以支持结果解释与过程回溯。基于上述需求，平台最终形成了以统一数据基础为支撑，集知识单元构建、知识增强分析、SEM 自动化表征和形貌预测于一体的科研辅助分析系统。

\subsection{系统总体架构}
\ensubsection{Overall System Architecture}

为满足上述需求，平台采用分层架构设计，整体划分为前端展示层、后端业务层、算法服务层和数据存储层四个层次。图~\ref{fig:5_1_platform_architecture} 给出了平台总体架构与层间交互关系。

前端展示层负责问题输入、图像上传、参数录入和结果展示，后端业务层负责请求调度、数据组织和结果回写，算法服务层集成第三章的知识单元构建与检索方法以及第四章的自动化形貌表征与形貌预测方法，数据存储层负责结构化记录和图结构知识的持久化。

该架构使样品记录、知识检索、SEM 自动化表征和形貌预测能够在同一流程中连续调用，并通过样品编号和来源信息实现结果关联与过程回溯。

\section{核心功能实现}
\ensection{Core Function Implementation}

平台围绕知识展示、实验数据管理、知识增强分析、SEM 自动化表征和形貌预测设置了五类核心功能模块，分别服务于知识浏览、样品管理、问题分析、图像测量和工艺预判等任务。

\subsection{知识库看板}
\ensubsection{Knowledge Base Dashboard}

知识库看板支持关键词检索、关系浏览和高频节点统计，为领域知识浏览与关系核查提供统一入口。输入关键词后，平台从图数据库调取相关节点、关系及来源证据，并以中心图谱和统计结果的形式返回，对应界面如图~\ref{fig:5_2_knowledge_dashboard}所示。

\subsection{实验数据库}
\ensubsection{Experimental Database}

实验数据库支持样品记录维护、SEM 图像上传绑定、工艺参数解析和分析结果回写，是平台各类分析任务的数据基础。系统以样品编号为主线组织样品信息、工艺参数、SEM 图像和分析结果，相关结构化数据统一保存在关系型数据库中，对应界面如图~\ref{fig:5_3_experimental_database}所示。

\ensubsection{Knowledge-enhanced Analysis Workbench}

\subsection{SEM 自动化表征工作台}
\ensubsection{SEM Automated Characterization Workbench}

SEM 自动化表征工作台支持图像上传、在线分割、结果核查和关键指标计算，是第四章方法的集成入口。上传 SEM 图像后，系统依次执行预处理、CNTSegNet-clDice 前景分割、骨架清理和指标提取，并输出覆盖密度、取向度、表观直径、平均曲率、波曲度和迂曲度 6 项核心形貌特征，同时附加骨架连接点数、连接点密度和骨架总长度等结构统计量，对应界面如图~\ref{fig:5_5_sem_workbench} 所示。

进一步拆分自动化表征链路可见，在 1920×1200 像素的实际 SEM 图像上，CNTSegNet-clDice 在 RTX 4060 上的平均推理时间为 0.31 s/张，在 CPU 上的平均推理时间为 2.74 s/张。传统阈值分割小于 0.01 s/张，单纯骨架化为 0.08 s/张，骨架分支清理为 2.78 s/张，因此自动化表征流程的主要耗时来自骨架清理与后续特征计算。

\section{小结}
\ensection{Summary}

本章完成了科研辅助分析平台的架构设计、运行环境配置、核心功能实现和功能测试。基于重复 10 次的功能测试与耗时统计可知，第三章和第四章提出的知识增强、自动化形貌表征和工艺—形貌关系分析方法已经在统一平台中完成集成部署，能够稳定支持样品管理、知识查询、SEM 自动化表征、形貌预测和结果解释等任务。
