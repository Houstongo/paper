\enchapter{Design and Implementation of a Scientific Research Assistant Analysis Platform for Carbon Nanotube Array Research}
\label{chap:platform_design}

\section{引言}
\ensection{Introduction}

前两章已经分别完成了面向 CNT 阵列研究的知识建模与检索增强、关键形貌自动化表征以及工艺—形貌关系分析。第三章对应全文分析链路中的知识支撑层，主要解决文献、实验记录和分析结果的组织与检索问题；第四章对应核心分析层，主要解决样品 SEM 图像中的关键形貌提取以及工艺参数驱动的关系分析问题。这些工作为样品分析、工艺比较和结果解释提供了方法基础。

但在实际研究中，知识查询、SEM 图像分析、工艺参数比较和结果解释通常是连续进行的。研究人员往往需要围绕同一样品或同一组工艺条件，同时参考实验数据、图像表征结果、领域知识和预测信息。若前述方法只作为独立算法或离线流程存在，虽然能够完成单项任务，却难以直接支撑完整的科研分析过程。因此，需要将这些方法组织到同一平台中，形成可直接服务 CNT 阵列研究的集成流程。

第五章的重点不在于提出新的分析方法，而在于对前文方法进行系统集成与验证。本章主要介绍科研辅助分析平台的设计与实现，包括平台架构、运行环境和核心功能页面，同时说明第三章和第四章的方法如何在平台中落地，并结合知识辅助分析、SEM 图像自动表征以及工艺参数预测与解释等场景对平台功能进行验证。

\section{系统设计与运行环境}
\ensection{System Design and Runtime Environment}

\subsection{系统需求分析}
\ensubsection{System Requirement Analysis}

面向碳纳米管阵列研究的科研辅助分析平台，其需求主要来自样品分析、工艺比较和结果解释等实际研究过程。首先，平台需要对样品信息、工艺参数、SEM 图像、知识单元和分析结果进行统一管理。碳纳米管阵列研究往往围绕同一样品开展多类分析，因此平台应以样品编号为主线组织多源数据，为后续模块调用提供一致的数据基础。其次，平台需要支持从文献、实验记录和历史分析结果到知识单元的构建与入库。碳纳米管阵列相关知识通常分散在不同来源中，只有先完成整理、构建和存储，后续知识增强分析才有稳定基础。再次，平台需要集成知识分析、SEM 图像表征和工艺参数预测三类核心任务，减少不同工具之间的切换，并保证各类分析结果能够相互关联。最后，平台还需要支持结果解释与回溯。无论是知识检索结果、图像表征指标还是预测结果，都应关联到具体样品、工艺条件和来源证据。基于这些需求，平台在统一数据基础上组织知识单元构建、知识检索、图像表征和预测解释等功能，形成面向碳纳米管阵列研究的专用分析系统。

\subsection{系统总体架构}
\ensubsection{Overall System Architecture}

为满足上述需求，平台采用分层架构设计，整体划分为前端展示层、后端业务层、算法服务层和数据存储层四个层次。图~\ref{fig:5_1_platform_architecture} 给出了平台总体架构与层间交互关系。

前端展示层提供统一交互入口，主要包括知识库看板、实验数据库、知识增强分析工作台、SEM 自动化表征工作台和形貌预测工作台，负责接收问题输入、图像上传和参数录入，并展示知识证据、形貌指标和预测结果。后端业务层负责组织具体业务流程，包括数据管理与分析、知识检索、知识单元构建、图像分析和机器学习预测等模块，用来连接前端请求与底层算法能力，并完成多模块结果组织。算法服务层对应平台中的核心分析方法，包括大模型接口、TCCER 算法、知识单元构建算法、SEM 表征算法和形貌预测算法。其中，第三章提出的方法主要部署在知识单元构建算法和 TCCER 算法中，第四章提出的方法主要部署在 SEM 表征算法和形貌预测算法中。数据存储层保存样品信息、SEM 图像、知识单元和预测结果等数据，为上层模块提供统一支撑。

如图~\ref{fig:5_1_platform_architecture} 所示，该架构在纵向上对应展示、调度、计算和存储，在横向上对应知识分析、图像表征和形貌预测三类功能链路。前端提交请求后，后端根据任务类型组织数据读取和算法调用，算法服务层完成具体计算，再将结果返回业务层并写入数据存储层。这样一来，知识单元构建、知识检索、SEM 图像表征和形貌预测等能力就被放在同一套系统中统一组织。

\section{核心功能实现与界面展示}
\ensection{Core Function Implementation and Interface Display}

\subsection{知识库看板}
\ensubsection{Knowledge Base Dashboard}

知识库看板用于展示平台中已构建的领域知识资源，是第三章知识建模与知识单元构建结果在平台中的集中可视化入口。研究人员可围绕具体术语或问题查看知识图谱中的节点关系、关系类型分布及高频关联内容，从而把握当前知识库的组织情况。

这个页面支持关键词检索、关系类别切换，以及图谱摘要和中心图谱展示。用户输入检索词后，平台可按工艺--形貌、形貌--性能、工艺--性能和机理证据等类别汇总相关关系，并在右侧以知识图谱形式展示节点及其连接关系。页面左侧给出关系数量统计和高频节点信息，便于快速定位核心概念及主要关联路径。

这个页面把分散的文献知识、实验知识和分析知识集中展示出来，方便后续继续做知识增强分析。知识库看板页面如图~\ref{fig:5_5_knowledge_dashboard} 所示，页面整体由图谱摘要区和知识图谱展示区构成。

\subsection{实验数据库}
\ensubsection{Experimental Database}

实验数据库页面主要用来管理样品基础信息、工艺参数、SEM 图像及相关分析结果。页面以样品编号为索引，把同一样品的实验记录、图像数据和结果数据对应起来，方便后续知识分析、图像表征和形貌预测。

该页面支持实验记录的新增、查询、修改和删除，可满足碳纳米管阵列样品数据的日常维护需求。对于 SEM 图像数据，系统支持 TIFF 和 PNG 等常用格式文件的拖拽上传与点击上传，并在上传过程中完成样品绑定与基础信息登记。对于实验命名中包含的工艺信息，系统还可结合预设正则表达式自动解析部分参数并写入记录，从而减少人工录入工作量。

除单条记录管理外，实验数据库页面还支持批量管理与报表导出。研究人员可围绕实验批次、样品编号或工艺条件进行筛选与整理，并将结构化实验记录导出为工艺报表，用于后续统计分析和实验归档。该页面在统一存储实验数据的同时，也为样品追溯、批量比较和结果整理提供了直接支撑。

实验数据库页面如图~\ref{fig:5_2_experimental_database} 所示。页面以样品记录列表形式组织实验编号、工艺参数、SEM 预览图及拍摄信息，能够较直观地反映样品数据的统一管理方式。

\subsection{知识增强分析工作台}
\ensubsection{Knowledge-enhanced Analysis Workbench}

知识增强分析工作台承接第三章提出的知识检索与分析方法，用于围绕科研问题开展知识辅助分析。用户可结合具体任务类型输入问题，平台完成查询解析、知识召回、关系扩展和证据组织，并输出与当前问题相关的分析结论和证据链结果。

该工作台提供历史对话区、任务示例区、模型与检索策略配置区以及分析输入区，可支持研究人员围绕工艺分析、机理解释和结果论证等任务快速发起知识增强分析。页面把问题输入、检索策略选择和模型调用集中在同一界面中，便于根据具体任务调整分析方式。

该工作台的输出结果不仅包含问题答案，还包含来源信息和关系链说明，因此更适合碳纳米管阵列研究中的工艺分析、形貌解释和结果论证任务。通过这一页面，领域知识能够直接参与科研分析过程。知识增强分析工作台页面如图~\ref{fig:5_4_knowledge_analysis_workbench} 所示，界面中集成了历史对话、任务示例和参数配置等区域。

\subsection{SEM 自动化表征工作台}
\ensubsection{SEM Automated Characterization Workbench}

SEM 自动化表征工作台对应第四章提出的图像自动分析方法，用于对样品 SEM 图像进行自动处理、结果展示与人工核查。用户上传图像后，平台完成预处理、前景分割、骨架分析和特征提取，并输出覆盖密度、取向度、表观直径、平均曲率、波曲度和迂曲度等关键形貌指标。

这个工作台不只给出数值结果，还把自动化表征结果直接可视化展示出来。页面会同时展示原始图像、分割结果、形貌指标和相关统计信息，方便研究人员直接检查算法输出是否与样品实际形貌一致。

该工作台还结合历史样品数据对当前结果给出可信度评分，用于反映自动化表征结果在已有数据分布中的稳定性与一致性。把结果可视化、人工核查和可信度评分放在一起后，平台既提高了自动化表征效率，也方便用户更稳妥地使用算法结果。SEM 自动化表征工作台页面如图~\ref{fig:5_3_sem_workbench} 所示，界面中同步展示了图像、指标结果和可信度信息。

\subsection{形貌预测工作台}
\ensubsection{Morphology Prediction Workbench}

形貌预测工作台用于将工艺参数与关键形貌指标之间的关系建模结果转化为可直接调用的平台功能。用户输入目标工艺参数后，平台调用预测模型输出相应形貌指标预测值，并联动知识增强分析模块补充解释信息。

该工作台集成模型选择、工艺参数录入、关键形貌指标展示、预测解释报告和近邻样品参考等区域。页面上方给出表观直径、覆盖密度、取向度、平均曲率、迂曲度和波曲度等指标的预测结果，中部输出面向当前工艺参数的解释报告，并结合知识增强分析结果补充相关机理说明。页面还可给出与当前参数条件相近的历史样品，用于辅助研究人员开展横向比较。

该工作台把预测结果和知识解释放在同一页面中，可为工艺比较、参数筛选和实验方案设计提供参考。平台不仅能够完成已有样品的分析，也能够支持面向工艺条件的前向预测与解释。形貌预测工作台页面如图~\ref{fig:5_6_morphology_prediction_workbench} 所示，界面中展示了参数输入、指标预测、解释报告和近邻样品信息。

\section{小结}
\ensection{Summary}

本章完成了科研辅助分析平台的架构设计、运行环境配置和核心页面实现，并对主要功能进行了测试。测试结果表明，前文提出的知识增强、自动化形貌表征和工艺--形貌关系建模方法已经可以在统一平台中部署，并能够支持样品分析、工艺比较和结果解释等任务。
